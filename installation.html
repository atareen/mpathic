Installation using pip:¶

The easiest way to install sortseq_tools is to download it from PyPi. You can do this using the command:

pip install sortseq

If you are the super user and need permission to download:

sudo pip install sortseq

Alternatively you can download the source code from github and unzip the file. Then open a terminal, move into the downloaded directory, and then execute the command:

python setup.py install

To install as superuser:

sudo python setup.py install

To install locally:

python setup.py install --user
