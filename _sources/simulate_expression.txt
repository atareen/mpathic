.. _simulate_expression:

==========================================
``simulate_expression``
==========================================

.. contents::

Overview
-------------
``simulate_expression`` is a program within the MPAthic package which simulates
an MPRA experiment or a dms_experiment. 

After you install `MPAthic`_, this program will be available to run at the command line. 

Command-line usage
---------------------
.. argparse::
   :module: MPAthic.sortseq_for_doc
   :func: parser
   :prog: MPAthic
   :path: simulate_expression

   

   
Examples
-----------
This module accepts a library file as input. The input data file must contain
a 'ct' column and a sequences column. If the sequences are DNA the
column should be labeled 'seq', if they are RNA, the column should be labeled
'seq_rna', and if they are amino acid sequences they should be labeled 'seq_pro'.

Example Input Data Table::

   ct    seq    
   13    ACTAG 
   5     AGGTA 
   ...

Example Output Table::

   ct    ct_0     ct_1   seq
   13    1        5      ACG
   28    8        5      GGT
   ...

Example Command to Run the analysis::
   
    sortseq simulate_expression -i library.txt -o simulated_experiment.txt 
--totallibcounts 10000 --totalmRNAcounts 10000
